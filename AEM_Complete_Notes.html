Unit 01

Electromechanical Systems & Robotics

1.1 What is a System?

A system is a set of interconnected components that work together to achieve a desired objective. Systems are classified based on their nature and how they interact with their environment.

Classification of Systems

  • Natural Systems — e.g., solar system, human body
  • Man-made Systems — e.g., power plant, computer
  • Open Systems — exchange matter/energy with environment
  • Closed Systems — no exchange with environment

1.4 Robotics: Fundamentals

Definition Robot: The word was coined by Czech novelist Karel Čapek in a 1920 play "Rossum's Universal Robots (RUR)." In Czech, "robot" means worker/servant.

Robotics: The term was introduced by Isaac Asimov in 1942 in his story "Runaround." It is the science of designing, manufacturing, and using robots.

Asimov's Laws of Robotics

  1. Zeroth Law: A robot may not injure humanity or, through inaction, allow humanity to come to harm.
  2. First Law: A robot may not injure a human being or, through inaction, allow a human to come to harm — unless this violates a higher-order law.
  3. Second Law: A robot must obey orders from humans, except where such orders conflict with a higher-order law.
  4. Third Law: A robot must protect its own existence, as long as such protection does not conflict with a higher-order law.

1.7 Robot Kinematics

Kinematics is the science of motion — studying positions, velocities, and accelerations of links and joints without considering forces.

Degrees of Freedom (DOF)

DOF is the number of independent displacements/motions a robot can perform. Each revolute or prismatic joint contributes 1 DOF. The human arm has 7 DOF (3 shoulder + 1 elbow + 3 wrist). The PUMA 560 robot has 6 DOF (all revolute).

Forward Kinematics vs Inverse Kinematics

Forward KinematicsInverse Kinematics
Given: link lengths + joint anglesGiven: link lengths + destination position
Find: end-effector position (coordinates)Find: joint angles needed to reach destination
Unique solutionMultiple solutions possible
Analogy: given road map → find destinationAnalogy: given destination → find the path

Reference Frames

Different parts of the robot operate in different coordinate frames. Types include:

  • World Frame — fixed reference frame
  • Joint Frame — attached to each joint
  • Tool Frame — attached to the end effector
  • Product Frame — attached to the workpiece

Transformation Matrices

Finding coordinates of a point with respect to different frames is called transformation. This involves rotation and translation matrices.

Rotation Matrix about Z-axis R_z(θ) = | cos θ -sin θ 0 | | sin θ cos θ 0 | | 0 0 1 |
Rotation Matrix about X-axis R_x(θ) = | 1 0 0 | | 0 cos θ -sin θ | | 0 sin θ cos θ |
Rotation Matrix about Y-axis R_y(θ) = | cos θ 0 sin θ | | 0 1 0 | |-sin θ 0 cos θ |

Homogeneous Transformation Matrix (4×4)

Combines rotation and translation into a single matrix. Also known as Denavit-Hartenberg (D-H) Matrix.

4×4 Homogeneous Transformation T = | R₃ₓ₃ d₃ₓ₁ | R = Rotation Matrix (3×3) | 0₁ₓ₃ 1 | d = Translation Vector (3×1) Bottom row = Scaling Factor

2.3 Hydraulic Actuators

Uses pressurized fluid (oil) to generate motion. Best for high-force applications.

Components of Hydraulic System

  1. Reservoir — Oil storage and conditioning
  2. Pump — Pressurizes oil and sends it to actuator
  3. Direction Control Valve (DCV) — Directs flow to actuator ports
  4. Flow Control Valve — Controls flow rate → controls speed
  5. Pressure Control Valve — Maintains required pressure; releases excess oil
  6. Actuator — Piston-cylinder (linear) or hydraulic motor (rotary)
Pascal's Law F = P × A Force = Pressure × Area of piston

2.4 Pneumatic Actuators

Uses compressed air. Fast operation, suitable for small robots and packaging.

  • Linear type — piston-cylinder arrangement
  • Rotary type — vane-type motors
  • Advantages: Fast, works in extreme temperatures
  • Disadvantages: Limited stroke, air leakage, moisture contamination, noisy

2.6 Electromagnetism — Key Concepts

Properties of Magnetic Lines of Force

  1. Invisible, imaginary, elliptical lines that never cross
  2. Travel from N → S (external) and S → N (internal)
  3. Parallel lines travelling in same direction REPEL
  4. Act like stretched rubber bands — always try to contract
  5. Prefer a path of minimum reluctance

Important Magnetic Quantities

Quantity
Formula / Unit
Permeability (μ)
H/m — ability to allow magnetic flux
MMF (F)
F = N × I (Amp-Turns)
Flux Density (B)
B = Φ/A (Tesla = Wb/m²)
Field Intensity (H)
H = NI/l (A/m)
Reluctance (S)
S = l/(μA) — opposition to flux

Faraday's Laws of Electromagnetic Induction

First Law: When magnetic flux linking a conductor changes, an EMF is induced.
Second Law: The magnitude of induced EMF is proportional to the rate of change of flux linkages.

E = −N (dΦ/dt) — the negative sign is due to Lenz's Law.
Lenz's Law The induced current flows in such a direction as to oppose the cause producing it (e.g., movement of magnet).

Motor Principle

"When a current-carrying conductor is placed in a magnetic field, it experiences a mechanical force." Direction given by Fleming's Left Hand Rule.

B-H Curve (Hysteresis Loop)

  • Sluggish start → Linear rise → Saturation
  • Retentivity — residual magnetism when H is removed
  • Coercivity — reverse H needed to demagnetize
  • Loop area ∝ energy loss per cycle

Grippers & End Effectors

3.1 What is a Gripper?

A gripper is a mechanical/robotic device that acts as the "hand" of a robot arm — enabling holding, tightening, handling, and releasing of objects. A gripper is a type of end effector, but not all end effectors are grippers.

Modes of Grasp

  • External Grip — fingers close around the outside of the object
  • Internal Grip — fingers expand inside a cavity of the object

Gripping Categories

Coarse GrippingRobot holds workpiece; precision not criticalStacking boxes, dipping castings

3.4 Vacuum Grippers

Use suction cups powered by vacuum generators (Venturi effect). Ideal for flat, smooth, non-ferrous surfaces.

Suction Force F = K × P × A where F = force, K = constant, P = vacuum pressure, A = cup area

Advantages

  • Require only one surface to grasp
  • Apply uniform pressure
  • Suitable for variety of materials
  • Low cost, easy maintenance

3.5 Magnetic Grippers

Use electromagnets to attract ferrous materials. The magnetic head has a ferromagnetic core with conducting coils.

Maxwell's Equation for Magnetic Force P ≥ (a + g) × m × FoS where a = acceleration, g = gravity, m = mass, FoS = Factor of Safety

Advantages

  • Fast pick-up
  • Tolerates size variations
  • Handles parts with holes
  • Needs only one surface

Disadvantages

  • Residual magnetism issues
  • May pick multiple parts from stack
  • Only works for ferrous materials

3.6 Adhesive Grippers

Grasp objects by literally sticking to them using adhesive surfaces (e.g., two-sided tape, micro-fiber adhesion). Advantage: keep adhesive sticky for long time without maintenance.

4.3 Zener Diode

A heavily doped PN junction diode designed to operate in the reverse breakdown region without damage. The reverse voltage (VZ) remains constant — ideal for voltage regulation.

Characteristics

  • Breakdown voltage range: 2.4V to 200V (up to 1kV)
  • Voltage tolerance: typically ±5%
  • Best temperature stability around 5V
  • Forward characteristics same as normal PN diode

Breakdown Mechanisms

Zener BreakdownAvalanche Breakdown
Due to heavy doping → high electric fieldDue to collision of free electrons with atoms
Occurs at low voltages (<5V)Occurs at higher voltages (>5V)
Electric field tears electrons from covalent bondsChain reaction of ionization

Applications

  1. Voltage Regulator: Maintains constant output voltage despite input variations
  2. Over-voltage Protection: Fuse + Zener combination protects circuits
  3. Clipping Circuits: Clips voltage at VZ level
  4. Voltage Shifter: Vout = Vin − VZ
Voltage Shifter Example V_in = 7V, V_Z = 3.3V V_out = V_in - V_Z = 7 - 3.3 = 3.7V

5.4 AREF (Analog Reference)

The reference voltage that the ADC uses to convert analog signals to digital values.

ADC Resolution Resolution = V_ref / 1024 For 5V: 5/1024 = 4.88 mV per step For 3.3V: 3.3/1024 = 3.22 mV per step

A lower AREF gives finer resolution for the same voltage range.

Arduino Analog Functions

int val = analogRead(pin);     // reads 0–1023 (10-bit)
analogWrite(pin, value);       // outputs PWM 0–255 (8-bit)
analogReference(EXTERNAL);     // use external AREF
analogReference(DEFAULT);      // use internal 5V

5.5 Interrupts

Polling vs Interrupt

PollingInterrupt
CPU repeatedly checks port at intervalsSignal triggers ISR automatically
Simple to codeMore complex to code
Inefficient (checks even when no change)Efficient (only when event occurs)
Predictable timingLess predictable timing

When Interrupt Occurs

  1. Main process pauses; return address saved to stack
  2. ISR (Interrupt Service Routine) executes
  3. Main process resumes from saved address

Pin Change Interrupts (PCINT)

ATmega328P has 3 PCINT groups sharing 3 vectors:

RegisterGroupPins
PCIE0 / PCMSK0PCINT0-7Port B (Digital 8-13)
PCIE1 / PCMSK1PCINT8-14Port C (A0-A5)
PCIE2 / PCMSK2PCINT16-23Port D (Digital 0-7)

Interrupt Registers

  • PCICR — Pin Change Interrupt Control Register (enables groups)
  • PCIFR — Pin Change Interrupt Flag Register (set when triggered)
  • PCMSKx — Pin Change Mask Register (selects specific pins)

Arduino Interrupt Example

const byte interruptPin = 3;
volatile byte state = LOW;     // 'volatile' for ISR variables

void blink() { state = !state; }

void setup() {
  pinMode(13, OUTPUT);
  pinMode(interruptPin, INPUT_PULLUP);
  attachInterrupt(
    digitalPinToInterrupt(interruptPin),
    blink, CHANGE
  );
}

void loop() {
  digitalWrite(13, state);
}
volatile keyword Directs the compiler to load the variable from RAM (not register cache). Essential for variables shared between ISR and main code.

NPN vs PNP Sensor Output

  • NPN (Sinking): Current "sinks" down to ground through the sensor's internal switch. Load connected between +V and output.
  • PNP (Sourcing): Current "sources" from +V through the sensor's switch to the load. Load connected between output and ground.

6.3 Infrared (IR) Sensors — Detailed

Fig 6.1 — IR proximity sensor: reflective mode

Three Types of IR Sensors

  1. Intensity-based: Simple reflective/break-beam; susceptible to ambient light
  2. Modulated IR (Proximity): Modulated signal; insensitive to ambient light
  3. IR Ranging (Distance): Uses position-sensitive detector; immune to color/reflectivity. Range: 10-80cm

6.4 Tactile Sensors

Require physical touching between sensor and object. Critical for assembly, pick-and-place, and grasping operations.

Capabilities

  • Presence detection, shape/location/orientation
  • Contact area pressure and distribution
  • Force magnitude, location, and direction
  • Surface texture monitoring, damage detection
  • Grasp verification, slip detection

Contact Sensors (Limit Switches)

Simple switches (NO = Normally Open, NC = Normally Closed) that detect when a moving object physically touches and forces contacts open or closed.
