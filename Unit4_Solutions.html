Question 01 5 Marks

Explain the concept of forward and reverse biasing of a diode with its characteristics.

PN Junction Diode

A diode is a two-terminal semiconductor device formed by joining P-type and N-type semiconductor materials. It allows current to flow primarily in one direction.

Forward Biasing

Forward Bias When the positive terminal of the battery is connected to the P-side (anode) and the negative terminal to the N-side (cathode), the diode is forward biased.
  • The applied voltage opposes and reduces the depletion region width.
  • When applied voltage exceeds the barrier potential (≈0.7V for Si, ≈0.3V for Ge), current starts flowing.
  • The diode acts as a closed switch — allows current flow.
  • Current increases exponentially with voltage beyond the knee voltage.

Reverse Biasing

Reverse Bias When the positive terminal is connected to the N-side and negative terminal to the P-side, the diode is reverse biased.
  • The applied voltage supports and increases the depletion region width.
  • Only a very small reverse saturation current (leakage current) flows due to minority carriers.
  • The diode acts as an open switch — blocks current flow.
  • If reverse voltage exceeds the breakdown voltage, a large reverse current flows (avalanche/zener breakdown).

Question 02 5 Marks

Elaborate on the working of a Bridge Rectifier with a circuit diagram and waveforms.

What is a Bridge Rectifier?

Definition A bridge rectifier is a full-wave rectifier that uses four diodes arranged in a bridge configuration to convert AC input into pulsating DC output. It utilizes both half-cycles of the AC input.

Working

Positive Half Cycle (A is positive, B is negative):

  • Diodes D1 and D3 are forward biased → conduct current.
  • Diodes D2 and D4 are reverse biased → do not conduct.
  • Current path: A → D1 → Load (top to bottom) → D3 → B
  • Current flows through the load in one direction.

Negative Half Cycle (B is positive, A is negative):

  • Diodes D2 and D4 are forward biased → conduct current.
  • Diodes D1 and D3 are reverse biased → do not conduct.
  • Current path: B → D2 → Load (top to bottom) → D4 → A
  • Current flows through the load in the same direction as before.

Fig 4.2 — Bridge Rectifier circuit diagram and waveforms

Key Formulas

Bridge Rectifier Formulas V_dc = 2Vm/π = 0.636 × Vm Ripple frequency = 2 × f (input frequency) PIV (Peak Inverse Voltage) = Vm Efficiency ≈ 81.2%

Advantages over Center-Tapped Full-Wave Rectifier:

  • No center-tapped transformer required — cheaper
  • PIV = Vm (vs 2Vm for center-tapped) — diodes need lower voltage rating
  • Better transformer utilization factor (TUF)

Question 03 5 Marks

Discuss the role of the Zener Diode as a voltage stabiliser.

What is a Zener Diode?

Definition A Zener diode is a specially designed PN junction diode that operates in the reverse breakdown region. Unlike ordinary diodes, it is designed to operate safely in breakdown — maintaining a nearly constant voltage across its terminals.

Working as a Voltage Stabiliser (Regulator)

The Zener diode is connected in reverse bias across the load. A series resistor (Rs) limits the current.

Operating Principle:

  1. When input voltage Vin > VZ (Zener voltage), the Zener diode enters breakdown and conducts.
  2. In breakdown, the voltage across the Zener (and hence the load) remains approximately constant at VZ, regardless of input voltage variations or load current changes.
  3. The excess voltage (Vin − VZ) is dropped across the series resistor Rs.
  4. If Vin increases, more current flows through the Zener (IZ increases) while VL stays at VZ.
  5. If Vin decreases (but stays > VZ), IZ decreases but VL still stays at VZ.
Key Equations V_out = V_Z (constant, as long as V_in > V_Z) I_s = (V_in - V_Z) / Rs I_s = I_Z + I_L I_L = V_Z / R_L

Question 04 5 Marks

Define α, β, and γ of a transistor. Establish the relation between α and β.

Definitions

α (Alpha) — Common Base Current Gain

Definition α (alpha) is the ratio of collector current (Ic) to emitter current (Ie) in Common Base (CB) configuration.
α = Ic / Ie
Value: Always < 1 (typically 0.95 to 0.99)

β (Beta) — Common Emitter Current Gain

Definition β (beta) is the ratio of collector current (Ic) to base current (Ib) in Common Emitter (CE) configuration.
β = Ic / Ib
Value: Always > 1 (typically 20 to 500)

γ (Gamma) — Common Collector Current Gain

Definition γ (gamma) is the ratio of emitter current (Ie) to base current (Ib) in Common Collector (CC) configuration.
γ = Ie / Ib
Value: γ = β + 1

Establishing the Relation between α and β

Derivation We know: Ie = Ic + Ib ... (1) [KCL at transistor] From definitions: α = Ic / Ie → Ic = α × Ie ... (2) β = Ic / Ib → Ib = Ic / β ... (3) Substituting (2) and (3) in (1): Ie = α × Ie + Ic/β Since Ic = α × Ie: Ie = α × Ie + (α × Ie)/β Dividing both sides by Ie: 1 = α + α/β 1 = α(1 + 1/β) 1 = α(β + 1)/β Therefore: α = β / (β + 1) Rearranging: α(β + 1) = β αβ + α = β α = β − αβ α = β(1 − α) β = α / (1 − α) Also: γ = β + 1 = 1 / (1 − α)

Question 05 5 Marks

Draw and explain the output characteristics of the BJT in CE configuration. Define β.

CE Configuration — Output Characteristics

The output characteristics plot Ic vs VCE for different values of base current Ib.

Fig 4.4 — CE Output Characteristics (Ic vs Vce for different Ib)

Three Regions of Operation:

  1. Cut-off Region: Both junctions reverse biased. Ib = 0, Ic ≈ 0. Transistor acts as an open switch.
  2. Active Region: BE junction forward biased, BC junction reverse biased. Ic = β × Ib. Transistor acts as an amplifier.
  3. Saturation Region: Both junctions forward biased. VCE ≈ 0.2V. Transistor acts as a closed switch.

Definition of β:

β Definition β (DC current gain in CE) = Ic / Ib β indicates how much the base current is amplified. If β = 100 and Ib = 10μA, then Ic = β × Ib = 100 × 10μA = 1 mA

Question 06 5 Marks

Explain a transistor as a switch with a neat circuit diagram and waveforms.

Transistor as a Switch

A transistor can be operated as a switch by driving it between cut-off (OFF state) and saturation (ON state), bypassing the linear active region.

OFF State (Cut-off):

  • Input voltage Vin = 0V (LOW) → Ib = 0
  • No collector current flows: Ic = 0
  • Full supply voltage appears across collector-emitter: VCE = VCC
  • Output = HIGH (VCC)
  • Transistor behaves like an open switch

ON State (Saturation):

  • Input voltage Vin = HIGH → sufficient Ib flows to saturate the transistor
  • Maximum collector current flows: Ic = VCC / RC
  • VCE ≈ 0.2V (saturation voltage, almost zero)
  • Output = LOW (≈0V)
  • Transistor behaves like a closed switch

Question 08 5 Marks

Describe FET as a voltage-controlled device with a suitable explanation.

FET — Voltage-Controlled Device

Key Concept In a FET, the drain current (ID) is controlled by the gate-to-source voltage (VGS), NOT by gate current. This makes FET a voltage-controlled device, unlike BJT which is current-controlled.

How it works (N-channel JFET):

  1. The FET has a channel (N-type or P-type) between Source and Drain through which current flows.
  2. The Gate terminal is connected to a PN junction that surrounds the channel.
  3. When VGS = 0V, the channel is fully open → maximum drain current (IDSS) flows.
  4. As VGS becomes more negative, the depletion region widens, narrowing the channel → ID decreases.
  5. At VGS = VP (pinch-off voltage), the channel is completely closed → ID = 0.
  6. Since the gate junction is reverse biased, practically no gate current flows (IG ≈ 0). The input impedance is extremely high (MΩ range).
Shockley's Equation (JFET) ID = IDSS × (1 − VGS/VP)² Where: IDSS = Maximum drain current (at VGS = 0) VP = Pinch-off voltage VGS = Gate-to-source voltage

Why "Voltage-Controlled"?

  • The output current (ID) is determined solely by the input voltage (VGS).
  • No input current is needed to control the output — the gate draws negligible current.
  • This is unlike BJT where base current (input current) is needed to drive collector current.
  • The gain parameter is transconductance (gm) = ΔID/ΔVGS (measured in mA/V or mS).

AND Gate using Transistors

An AND gate produces a HIGH output only when ALL inputs are HIGH. Using NPN transistors, two transistors are connected in series.

Fig 4.6 — AND gate using two NPN transistors in series

Working:

  • A=0, B=0: Both Q1 and Q2 are OFF (cut-off). No current through Rc. Y = VCC → But wait! — Actually, since we need output LOW when not both HIGH, the output is taken differently. Let me clarify:

In the series transistor AND gate, both transistors must be ON for current to flow through Rc and pull the output LOW. The output is inverted (NAND behavior). To get AND, the output is typically taken after an additional inverter stage, or the output logic is interpreted inversely.

Question 10 5 Marks

Numerical: β = 75, Ib = 40 μA (CE). Find Ic, Ie, α and γ.

Given

β = 75
Ib = 40 μA = 40 × 10⁻⁶ A

Solution

Step 1: Collector Current (Ic) Ic = β × Ib Ic = 75 × 40 × 10⁻⁶ Ic = 3000 μA = 3 mA
Step 2: Emitter Current (Ie) Ie = Ic + Ib Ie = 3000 + 40 Ie = 3040 μA = 3.04 mA
Step 3: Alpha (α) α = Ic / Ie α = 3000 / 3040 α = 0.9868 OR: α = β / (β + 1) = 75/76 = 0.9868 ✓
Step 4: Gamma (γ) γ = Ie / Ib γ = 3040 / 40 γ = 76 OR: γ = β + 1 = 75 + 1 = 76 ✓

Answers

Ie3.04 mA
α0.9868
γ76

Question 11 5 Marks

Numerical: Ie = 2.5 mA, Ic = 2.4 mA (CE). Find α, β, γ and Ib.

Given

Ie = 2.5 mA
Ic = 2.4 mA

Solution

Step 1: Base Current (Ib) Ib = Ie − Ic Ib = 2.5 − 2.4 Ib = 0.1 mA = 100 μA
Step 2: Alpha (α) α = Ic / Ie α = 2.4 / 2.5 α = 0.96
Step 3: Beta (β) β = Ic / Ib β = 2.4 / 0.1 β = 24 Verify: β = α / (1 − α) = 0.96 / 0.04 = 24 ✓
Step 4: Gamma (γ) γ = Ie / Ib γ = 2.5 / 0.1 γ = 25 Verify: γ = β + 1 = 24 + 1 = 25 ✓

Numerical: α = 0.995, Ic = 2 A (Common Base). Find Ie, Ib and β.

Given

α = 0.995
Ic = 2 A

Solution

Step 1: Emitter Current (Ie) α = Ic / Ie Ie = Ic / α Ie = 2 / 0.995 Ie = 2.0101 A ≈ 2.01 A
Step 2: Base Current (Ib) Ib = Ie − Ic Ib = 2.0101 − 2 Ib = 0.0101 A ≈ 10.05 mA
Step 3: Beta (β) β = α / (1 − α) β = 0.995 / (1 − 0.995) β = 0.995 / 0.005 β = 199 Verify: β = Ic / Ib = 2 / 0.01005 = 199 ✓
